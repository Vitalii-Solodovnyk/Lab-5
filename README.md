Lab 5 — Zadanie 1


Opis
Program wczytuje jeden wiersz tekstu (do momentu naciśnięcia Enter) i zlicza:
- DUŻE — wielkie litery (A..Z)
- MAŁE — małe litery (a..z)
- CYFRY — cyfry (0..9)
- INNE — wszystkie inne znaki (np. spacje, znaki interpunkcyjne itp.)


Znak Enter nie jest uwzględniany w statystykach.
